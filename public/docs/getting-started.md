# 快速开始指南

欢迎使用富文本编辑器！本指南将帮助您快速上手。

## 第一步：安装

首先，您需要安装项目依赖：

```bash
pnpm install
```

这将安装所有必需的包和依赖项。

## 第二步：启动开发服务器

安装完成后，运行开发服务器：

```bash
pnpm dev
```

服务器将在 `http://localhost:5173` 启动。

## 第三步：开始编辑

打开浏览器访问开发服务器地址，您将看到编辑器界面。

### 基本操作

编辑器提供了直观的工具栏，包含以下功能：

- **文本样式**：粗体、斜体、下划线
- **标题**：从 H1 到 H6
- **列表**：有序和无序列表
- **链接**：插入超链接
- **图片**：上传或插入图片
- **代码**：代码块和行内代码

### 快捷键

掌握快捷键可以大大提高编辑效率：

| 快捷键 | 功能 |
|--------|------|
| Ctrl+B | 加粗 |
| Ctrl+I | 斜体 |
| Ctrl+K | 插入链接 |
| Ctrl+S | 保存 |

## 第四步：保存和导出

编辑完成后，您可以：

1. 保存内容到本地存储
2. 导出为 Markdown 格式
3. 导出为 PDF 文档
4. 导出为 Word 文档
