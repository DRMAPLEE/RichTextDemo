# 文本对比编辑器 - 快速开始

## 🎯 一句话说明

在同一个视图中看到新旧文本的所有差异，点击按钮决定接受或拒绝每个更改。

## 🚀 30秒上手

1. **访问页面**：首页 → 点击"文本对比 🆕"
2. **查看差异**：滚动查看所有用彩色标记的变更块
3. **做出决定**：点击每个变更块下方的 ✓ 接受 或 ✗ 拒绝
4. **获取结果**：在右侧查看最终结果，点击"复制结果"

## 🎨 颜色含义

| 颜色 | 含义 | 说明 |
|------|------|------|
| 🔴 红色 | 删除 | 将被删除的旧内容 |
| 🟢 绿色 | 新增 | 将被添加的新内容 |
| 🔵 蓝色边框 | 待处理 | 等待你做决定 |
| ✅ 绿色边框 | 已接受 | 已接受，会应用到结果 |
| ⬜ 灰色 | 已拒绝 | 已拒绝，不会应用 |

## 🎮 操作说明

### 单个变更
```
每个变更块底部有两个按钮：
┌──────────────────────────┐
│ [✓ 接受更改] [✗ 拒绝更改] │
└──────────────────────────┘
```

### 批量操作
```
顶部工具栏：
- ✓ 接受全部：一键接受所有变更
- ✗ 拒绝全部：一键拒绝所有变更
- 🔄 重置示例：恢复默认示例
- 📋 复制结果：复制最终文本
```

### 编辑文本
```
右侧侧边栏：
- 点击 🔓 可以编辑
- 点击 🔒 锁定为只读
- 编辑后自动重新计算差异
```

## 💡 典型工作流

### 场景 1：代码审查
```
1. 将原代码粘贴到"原始文本"
2. 将修改后的代码粘贴到"新文本"
3. 在主视图中逐个审查每处修改
4. 接受好的修改，拒绝不合适的
5. 复制最终结果
```

### 场景 2：AI 建议审核
```
1. 左侧显示 AI 给出的所有改进建议
2. 在完整上下文中理解每个建议
3. 选择性地接受有价值的建议
4. 获得优化后的代码
```

### 场景 3：文档修订
```
1. 加载原始文档和修订版
2. 查看每一处文字改动
3. 决定保留哪些修订
4. 得到最终定稿
```

## 🔥 专业技巧

1. **保持上下文**：滚动查看时，周围未变更的文本帮助你理解变更的上下文
2. **灵活决策**：可以随时改变决定，只需再次点击按钮
3. **实时预览**：右下角"最终结果"实时显示效果，确保符合预期
4. **批量处理**：如果确定要全部接受/拒绝，使用顶部批量按钮更快

## 🆚 与传统对比的优势

| 传统方式 | 本工具 |
|---------|--------|
| 左右分栏，来回对照 | 内联显示，一目了然 |
| 整体接受或拒绝 | 逐块选择，精细控制 |
| 难以保持上下文 | 完整上下文，不会迷失 |
| 需要外部工具 | 浏览器直接使用 |
