# ToastUI 富文本编辑器项目总结

## 🎯 项目概述

本项目成功创建了一个基于 **ToastUI Editor** 的 WYSIWYG 富文本编辑器，使用 Vue 3 和 Tailwind CSS 构建。编辑器默认以 WYSIWYG 模式运行，提供所见即所得的编辑体验。

## 🔧 技术实现

### 核心技术栈

- **Vue 3** - 前端框架，使用组合式 API
- **ToastUI Editor 3.x** - 富文本编辑器核心
- **Tailwind CSS** - 样式框架
- **Vite** - 构建工具
- **Vue Router** - 路由管理

### 集成插件

- **代码语法高亮插件** - 使用 Prism.js
- **颜色语法插件** - 支持文本和背景色

### 编辑器配置

```javascript
{
  initialEditType: 'wysiwyg',        // 默认 WYSIWYG 模式
  height: '500px',                   // 编辑器高度
  previewStyle: 'vertical',          // 预览样式
  plugins: [                         // 插件配置
    [codeSyntaxHighlight, { highlighter: 'Prism' }],
    [colorSyntax]
  ],
  toolbarItems: [                    // 完整工具栏
    'heading', 'bold', 'italic', 'strike',
    'divider', 'hr', 'quote',
    'ul', 'ol', 'task', 'indent', 'outdent',
    'table', 'image', 'link',
    'code', 'codeblock', 'fullScreen'
  ]
}
```
